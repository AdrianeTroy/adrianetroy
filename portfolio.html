https://github.com/AdrianeTroy/RVMSchedSys

This project is a web-based scheduling system designed for Ramon V. Mitra Sports Complex and is still under development for educational purposes. It is a dynamic web page made using Laravel PHP framework see Laravel.

This web app was made for our 4th year thesis in BSIT S.Y 2021-2022. The main idea was to create a way for athletes and non-athletes to request schedules of a facility they want to reduce the hectic traditional way of pen and paper.
